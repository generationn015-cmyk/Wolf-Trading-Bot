### Key Insight: Why CSS Animations Keep Failing in Next.js App Router
- styled-jsx (`<style>{...}</style>` in JSX body) requires `styled-jsx` npm package
- Without it, Next.js App Router silently drops the style tag
- SVG renders frozen — no error, no warning
- Correct pattern: inject CSS via `document.head.appendChild(style)` in useEffect
- Classes applied via `className` on SVG elements work exactly like Tailwind
